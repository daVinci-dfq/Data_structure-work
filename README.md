## 题目要求

问题描述：
使用等价类来构造一个 `N * N` 的从左上角到右下角只有一条路径的随机迷宫，然后在这一迷宫上寻找迷宫路径。

该设计共包含如下四个部分：

1. 等价类数据结构的设计和实现
2. 构建随机迷宫
3. 寻找迷宫路径
4. 将迷宫和路径用图形方式画出 
   用图形方式将上述算法获得的随机迷宫及其上的最短路径画出。用线段来表示迷宫中的墙，用在每个方格中心的点来表示路径。
